# 🥗 תזונה חכמה — מעקב משקל וקלוריות

אפליקציית PWA (עברית, RTL) למעקב אחר משקל, קלוריות ומאקרו־נוטריאנטים, עם **זיהוי אוכל מתמונה** בעזרת בינה מלאכותית (Claude Vision).

קובץ יחיד, ללא שרת וללא תהליך build — פותחים את `index.html` ומתחילים. כל הנתונים נשמרים מקומית בדפדפן (`localStorage`).

## יכולות

- **פרופיל אישי ויעד חכם** — מחשב BMR/TDEE (נוסחת Mifflin-St Jeor) ויעד קלורי יומי לירידה במשקל לפי מין, גיל, גובה, משקל, רמת פעילות וקצב ירידה רצוי.
- **מעקב יומי** — טבעת התקדמות לקלוריות, פירוק חלבון/פחמימה/שומן, וקלוריות שנותרו ליום.
- **📷 זיהוי אוכל מתמונה** — מצלמים את הצלחת, וה־AI מזהה את המאכלים ומעריך גרמים, קלוריות וערכים תזונתיים.
- **🔍 מאגר מאכלים** מובנה (ערכים ל־100 גרם) להוספה מהירה לפי כמות.
- **✏️ הוספה ידנית** של כל מאכל.
- **⚖️ מעקב משקל** — רישום שקילות, גרף מגמה, BMI, אחוז התקדמות ליעד.
- **📅 יומן** — סיכום צריכה יומית והיסטוריה.
- **מצב כהה**, ייצוא/ייבוא גיבוי, ואיפוס.

## הפעלת זיהוי מתמונה (AI)

התכונה דורשת מפתח Anthropic API:

1. הפיקו מפתח ב־[console.anthropic.com](https://console.anthropic.com).
2. באפליקציה: **הגדרות → זיהוי אוכל מתמונה (AI)** → הדביקו את המפתח ושמרו.
3. בחרו מודל: `claude-opus-4-8` (מדויק, מומלץ) או `claude-sonnet-4-6` (מהיר/זול יותר).

**כיצד זה עובד:** האפליקציה מקטינה את התמונה ושולחת אותה ישירות מהדפדפן ל־Messages API של Anthropic, עם פלט מובנה (JSON Schema) המבטיח החזרת רשימת פריטים עם ערכים תזונתיים.

> **פרטיות ואבטחה:** הקריאה ל־API מתבצעת ישירות מהדפדפן (`anthropic-dangerous-direct-browser-access`), והמפתח נשמר רק במכשיר. זהו דפוס המתאים ל**שימוש אישי**. לפריסה ציבורית מומלץ להעביר את קריאת ה־API דרך proxy בצד שרת כדי לא לחשוף את המפתח.

## הפעלה

פתחו את `food-tracker/index.html` בדפדפן, או הגישו את התיקייה כ־static site. ניתן להתקין כאפליקציה (Add to Home Screen) הודות ל־`manifest.json`.

## הערות

- ערכים תזונתיים מהתמונה הם **הערכה** — לכל פריט מוצגת רמת ודאות (גבוהה/בינונית/הערכה).
- אין כאן ייעוץ רפואי. בכל שינוי תזונתי משמעותי התייעצו עם איש מקצוע.
